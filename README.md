### Правила гри

* Треба **керувати стрілками** і тікати від **червоного кола перехоплювача**, збираючи золоті монети.
* Гра триває **30 секунд**, за цей час потрібно набрати найбільше очок.
* **УВАГА:** Втікай від червоного перехоплювача, який **набирає швидкість**, чим більше монеток ти збереш.
* **Як почати?** Натисни на **[Пробіл]** у меню.

### Зберігання Прогресу

* Після кожної гри твій результат буде **оновлюватись і зберігатись** у `локал сторедж` (Local Storage).

1.  **Пауза!** Треба додати клавішу **[P]**, щоб можна було зупинити гру.
2.  **Кращий вигляд:** Хочу додати якісь **ефекти або анімацію**.
